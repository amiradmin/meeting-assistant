# 🎙️ Meeting Assistant

**Offline AI-powered meeting transcription & summarization tool** | Django + React + Whisper + Ollama

[![Docker](https://img.shields.io/badge/Docker-Ready-blue.svg)](https://docker.com)
[![License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)

## ✨ Features

- 🎤 Record or upload meeting audio
- 📝 Automatic speech-to-text transcription (Whisper)
- 🤖 Smart meeting minutes generation (Llama 3.2)
- 🔒 **100% offline** - No internet required, complete privacy
- 🐳 Easy setup with Docker

## 🚀 Quick Start

```bash
git clone https://github.com/yourusername/meeting-assistant.git
cd meeting-assistant
docker-compose up -d
